模糊搜索:

"key$":"SQL搜索表达式" => "key$":["SQL搜索表达式"]，任意SQL搜索表达式字符串，如 %key%(包含key), key%(以key开始), %k%e%y%(包含字母k,e,y) 等，%表示任意字符



```
{
    "[]": {
        "count": 20,
        "Moment": {
            "@column": "id,userId,content",
            "content$":"%APIJSON%"
         //   "@group": "userId"
        }
    }
}             

```

```
SELECT id,userId,content FROM test.Moment WHERE  (  (content LIKE '%APIJSON%')  )  LIMIT 20
```
